# Library Management System

Bu proje, **C# Windows Forms** ve **SQL Server** kullanılarak geliştirilmiş bir **Kütüphane Yönetim Sistemi**dir.  
Amaç, kütüphane içindeki kitap ve ödünç alma işlemlerinin dijital ortamda yönetilmesidir.

---

## Sistem Özellikleri
- Kitap listeleme ve arama
- Kitap ödünç alma ve iade işlemleri
- Kullanıcı ve rol yönetimi
- Raporlama işlemleri

---

## Kullanıcı Rollerİ
- **Öğrenci:** Kitap arama, ödünç alma, iade
- **Personel:** Ödünç taleplerini onaylama
- **Admin:** Kitap ve rapor yönetimi

---

## Kullanılan Tasarım Desenleri
- **State Pattern:** Ödünç alma sürecindeki durum geçişlerini yönetir
- **Strategy Pattern:** Farklı kitap arama yöntemlerini sağlar

---

## Kullanılan Teknolojiler
- C# (.NET WinForms)
- SQL Server

---

## Kurulum
1. SQL Server’da veritabanını oluşturun  
2. Gerekli tabloları ekleyin  
3. Connection String’i güncelleyin  
4. Projeyi Visual Studio’da çalıştırın  

---

## Notlar
- Veritabanı bağlantısı aktif olmalıdır  
- Uygulama masaüstü (Windows) ortamı için geliştirilmiştir
